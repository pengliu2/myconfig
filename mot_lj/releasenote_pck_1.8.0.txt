PCK, the abbreviation for Platform Creation Kit, Is a GUI tool, which helps developer to develop embedded programs. 
 
PCK shall provide the functionalities, Such as components dependency checking, Installation order determination, Building from source, Picking up the base lined distributions, Generating the root file system and creating the image, etc. 
 
For iDEN Samba Product,PCK could also be mentioned as Integration Tool, Since it covers a part of PCK functionalities, And focuses on the components integration.
 
What's New
---------------
  - Support of NOPASSWD option for sudo privilege.
  - Same version package reinstallation.
  - Environment checking at starting up.
  - Show recent projects in main menu.
  - Enable image generate in user mode RootFS.
  - Command line prelink interface.

[- CHANGE LOGS -]
* Support of NOPASSWD option for sudo privilege.
  - No password needed when user's id has NOPASSWD option in /etc/sudoers
  - Sudo password can be saved safely even if PCK is closed.

* Package reinstallation.
  - Highlight one or more installed pakcage, then reinstallation can
  be established by menu entry.

* Environment checking at starting up.
  - PCK can determine whether system environment can satisfy
  requirement of PCK.

* Show recent projects in main menu. 
  - Latest 9 projects user opened successfully will be listed in menu
  Project->Recent Project

* Enable image generate in user mode RootFS.
  - After switch to user mode to make install, it is unnecessary to
  switch back to normal mode in order to generate image.

* Command line prelink interface.
  - User can do prelink on his own RootFS whose Root is rootfspath.
  - The script interface:
  sudo createimage.sh prelink <arch> <rootfspath>
